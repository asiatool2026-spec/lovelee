🔑 인증 설정 (.env)

📋 Cookie 값 가져오는 방법

  1. 엘박스 접속 & 로그인
    lbox.kr 또는 research.lbox.kr 에서 로그인하세요.
  2. 개발자 도구 열기
    키보드에서 F12 키를 누르세요. (Mac은 Command + Option + I)
  3. Network 탭 클릭
    상단 탭 중 Network (네트워크) 를 클릭하세요.
  4. 아무 판례나 클릭
    엘박스에서 아무 판례나 눌러보세요. Network 탭에 목록이 채워집니다.
  5. 요청 항목 선택
    목록에서 api 또는 references 가 포함된 항목을 클릭하세요.
  6. Cookie 복사
    오른쪽에서 Headers 탭 → Request Headers 아래 cookie: 항목 전체를 드래그해서 복사하세요.
  7. 여기에 붙여넣기
    아래 텍스트박스에 붙여넣고 설정 저장 버튼을 누르면 완료입니다.

⚠️ Cookie 값은 로그인 정보와 같습니다. 타인과 공유하지 마세요. 로그아웃하거나 일정 시간이 지나면 만료됩니다.
